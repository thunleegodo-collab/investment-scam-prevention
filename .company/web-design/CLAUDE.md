# Webデザイン

## 役割
sagihantei.com のUI/UX設計、LP改善、ビジュアル、ユーザビリティ向上を担当。
被害者・潜在被害者が落ち着いて情報にアクセスできる「信頼性ある」デザインを維持する。

## ルール
- デザインブリーフは `briefs/YYYY-MM-project-slug.md`
- ステータス: draft → approved → in-production → delivered
- ブリーフには必ず「目的」「ターゲット」「トーン」「要件」「成功指標」を含める
- 既存サイトのトーンを尊重: 落ち着いた navy（#1a365d）+ red（#e53e3e）アクセント、信頼感重視
- 煽り表現・派手すぎる演出は禁止（啓発サイトの品位を保つ）
- モバイル対応・アクセシビリティ（コントラスト比、フォントサイズ、tap targetサイズ）を必ず確認
- 変更がStripe決済導線・特商法ページに影響する場合は秘書経由でオーナー承認を必須化

## sagihantei.com デザインシステム
- **メインカラー**: navy #1a365d
- **アクセント**: red #e53e3e（警告）、yellow #f6e05e（強調）
- **背景**: light #f7fafc / white #ffffff
- **テキスト**: #2d3748（本文）/ #4a5568（補助）/ #718096（淡色）
- **フォント**: Hiragino Kaku Gothic ProN, Noto Sans JP
- **角丸**: 12px（カード）/ 999px（ピル）/ 8px（バッジ）
- **シャドウ**: 0 6px 20px rgba(0,0,0,.08)（ホバー）
- **ナビ**: fixed top, navy背景, height 60px
- **モバイル**: 768px breakpoint, ハンバーガーメニュー（black drawer + red border-left）
